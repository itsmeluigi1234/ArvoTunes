# ArvoTunes

Discord Music Bot (Render Deployment)

A simple Discord music bot that plays music directly from YouTube. Built with Python, discord.py, youtube_dl, and Flask for 24/7 uptime on Render.


---

Features

Join and leave voice channels

Play music from YouTube URLs

Pause, resume, and stop music

Flask server keeps bot alive on Render

No Lavalink or Wavelink required

Mobile-friendly deployment



---

Requirements

Python 3.10+

Discord bot token

Render account (or similar cloud host)
